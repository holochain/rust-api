fetches the set of EntityAttributeValues that match constraints None = no constraint Some(Entity) = requires the given entity (e.g. all a/v pairs for the entity) Some(Attribute) = requires the given attribute (e.g. all links) Some(Value) = requires the given value (e.g. all entities referencing an Address)
